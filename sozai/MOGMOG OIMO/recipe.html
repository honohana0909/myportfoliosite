さつまいもの下処理方法

このさつまいもの下処理方法を覚えれば、さつまいもの扱い方が上手になります。これから紹介するレシピの１番初めに行ってほしい作業です。

  1. さつまいもについている泥や砂を流水できれいに洗い流す。
  2. 用途に合わせて切る。皮を蒸す場合はピーラーがおすすめ！
  3. 水を張ったボウルに５分程度さらしてアク抜きを行う。
  4. 水気を切れば、準備完了！

この簡単な４ステップを行うことでさつまいものおいしさを最大限に引き立たせることができます。

ー調理手順ー

  1. 鶏もも肉を１口大に切って、１０分程度料理酒に着けておく。
  2. 下処理したさつまいもを耐熱容器に入れレンジ５００Wで３分加熱。
  3. フライパンでサラダ油を熱し、鶏肉を中火で色がほんのり変わるまで焼く。
  4. さつまいもを加え鶏肉とさつまいもに焼き目をつける。
  5. 材料の🅰をすべて加え、煮からめる。
  6. 材料🅰の水分がある程度飛んだら火を止めて、お皿に盛りつけて、黒ゴマを散らしたら完成！

ー調理手順ー

  1. 下処理後のさつまいも、玉ねぎ、バター、水を鍋に入れて火にかける。このとき、沸騰までは強火、その後蓋をし、弱火にして２０分程度。
  2. フォークでつぶせる程度になったら別の容器に移してブレンダーやミキサーにかける。
  3. 鍋に戻し、牛乳（豆乳）と刻んだ固形ブイヨンを加え、ブイヨンが解けるまで弱火で伸ばす。
  4. 沸騰する直前で火をとめ、味をみて物足りないときは塩を加え、スーツカップに注ぐ。パセリを散らせば、完成！

ー調理手順ー

  1. 皮をむいて下処理をしたさつまいもを耐熱皿にいれてふんわりとラップをかけ、レンジ５００Ｗで８～１０分。竹串がスーッと通るぐらい。
  2. マッシャーでつぶす。（さらに、裏ごしをするとなめらかになる。）
  3. さつまいもが熱いうちにバター砂糖を加える。（さつまいもが覚めてしまった場合は小さじ１の水を加え、レンジ５００Ｗで１分加熱。）
  4. 牛乳（豆乳）を少しずつ加え、形が保てるぐらいの滑らかさになっればＯＫ。
  5. アルミカップやクッキングシートに整形し、卵黄を塗り、黒ゴマを散らす。
  6. 余熱あり２００℃のオーブンで１０～１５分または、トースターで５～１０分、焼き目が付く案で焼いたら完成！

生クリーム

ー調理手順ー

  1. 皮をむいて下処理をしたさつまいもを耐熱皿にいれてふんわりとラップをかけ、レンジ５００Ｗで８～１０分。竹串がスーッと通るぐらい。
  2. 裏ごしを２回する。
  3. 卵黄と砂糖をハンドミキサーで、ふわふわと白っぽくなるまで撹拌する。。
  4. ミキサーにすべての材料をいれ、なめらかにする。
  5. バットやアルミのボウルに移し替え、ラップをぴったりとする。その上から保冷剤を置いて冷凍庫へ入れる。
  6. ３０分から１時間おきに凍ってきたところを混ぜる。これを３回から５回繰り返す。
  7. ２時間以上冷やし固める。
  8. お皿に盛り付け、完成！ここで先ほど紹介した甘露煮のトッピングをすることもお勧め！

焼き芋編

オーブンでねっとり焼き芋

ー材料ー

さつまいも
小さめを入るだけ！

ー調理手順ー

  1. 下処理のStep１だけを行い、オーブンの用の天板の上にクッキングシートを敷き並べる。
  2. 余熱なしのオーブン１７０℃で３０分～４０分加熱する。
  3. 一度取り出し、フォークや竹串で数か所刺す。
  4. ミキサーにすべての材料をいれ、なめらかにする。
  5. 今度は余熱なし１６０℃のオーブンで９０分加熱する。
  6. 加熱し終えたら、完成！ここでさらにおいしく食べたい人は次のステップへ！！
  7. 加熱し終えても、オーブンから取り出さず、自然とオーブン内の温度が冷めるのをひたすら待つ。
  8. ３０～１時間たったら熟成された焼き芋の完成！
